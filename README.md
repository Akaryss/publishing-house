# Система учёта книг в издательстве

25 вариант лабораторной работы №1 по ИПП

Сущности:
- Книга (title, author, publisher, year)
- Автор (name, publicationsCount)

- Поиск всех книг конкретного издательства
- Показать авторов с количеством публикаций ≥ N
- Консольное меню
